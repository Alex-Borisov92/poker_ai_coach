# Study Plan Protocol

Study Plan is AI-generated from stats, leaks, selected hands, and coaching principles.

Required flow:

1. Pick top 1 to 3 focus areas.
2. Pick 5 to 10 hands only if hand IDs are available.
3. Create practical drills for one week.
4. Add a short checklist.
5. Include confidence and missing data.

Good drills:

- 3bet defense filter review.
- River bluff-catcher review.
- VPIP/PFR gap review: flats, limps, missed iso raises.
- Tournament cluster story review.
- Big-pot value target review.

Keep the plan practical and short.
